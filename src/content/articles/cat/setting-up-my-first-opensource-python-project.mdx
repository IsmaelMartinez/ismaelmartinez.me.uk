Aquest és l'inici d'una sèrie que documenta la creació del meu primer projecte a PyPI. Reconec que això representa tant la meva primera contribució al codi obert com la meva primera incursió en Python, de manera que hi ha un marge considerable de millora.

## Visió General del Projecte

El projecte s'anomena **logger-to-kibana** i se centra en principis d'automatització. L'eina analitza els loggers del codi i genera visualitzacions de Kibana a partir dels missatges de registre trobats durant l'execució.

## Funcionalitats Principals

El projecte inclou tres ordres principals:

- `process` -- processa una carpeta
- `process_and_generate` -- processa una carpeta i genera visualitzacions
- `process_generate_and_send` -- executa el pipeline complet incloent-hi la transmissió a Kibana

## Instal·lació

Els usuaris poden instal·lar les dependències mitjançant:

```bash
pip install -r requirements.txt
```

I accedir a la documentació d'ajuda executant:

```bash
python main.py
```

## Configuració

La configuració per defecte es troba a `settings.ini`, amb la possibilitat de personalització a través de la variable d'entorn `LOGGER_TO_KIBANA_CONFIG`.

Això us permet configurar:

- L'URL del punt d'accés de Kibana
- Les credencials d'autenticació
- Les plantilles i valors per defecte de les visualitzacions
- Els patrons d'anàlisi de registres
